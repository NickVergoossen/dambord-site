Mei

In maart heb ik met behulp van meneer Clijsters de code om het bord te laten. Deze toepassing is korter maar voor mij persoonlijk minder duidelijk omwille van een verschillende manier van programmeren. Dit wilt niet zeggen at ik dit niet ga gebruiken ik ga deze voor mij zelf en andere verduidelijke doormiddel van commentaar. Ook heb ik deze maand een programma geschreven voor het bewegen van stukken dit programma werkt tot nu toe niet maar hier word nu aan gewerkt.


April

In april ben ik begonnen met een kleine pre-presentatie voor de gip jury deze is opzicht vrij goed verlopen. Ook ben ik deze maand pas begonnen met het maken van mijn website en de blog die u nu momenteel leest.


Maart

Bij het begin van maart heb ik een programma geschreven die het gehele bord kan laten oplichten met het patroon van een dambord. Ook ben ik deze maand begonnen met het maken van het fysieke gedeelte van het bord deze heb ik nog voor de examens af weten te krijgen.


Februari

Februari ben ik begonnen met een test programma te schrijven voor het dambord. Deze was gebasseerd op een code die was geschreven in c++. Later bleek toch dat de code die ik al gedeeltelijk had geschreven niet zou werken omwille van bepaalde begrippen binnen c++ die ik nog niet begrijp.


Januari

In januari ben ik begonnen met het maken van een programma waarmee ik een enkele led kan aansturen inplaatst van de gehele strip. Ik heb ook gekeken naar de code van online dambord spelen zodat ik hier later misschien zelf een code mee kan maken.


December

In december heb ik een programma gemaakt waarmee ik de ledstrip kan aanzetten. ik heb daarna vervolgens ook een programma gemaakt waarbij ik de ledstrip kan aansturen aan de hand van mijn joystick.


November

in November heb ik informatie gekregen over hoe ik de blog moest gaan maken. Ook heb ik een programma geschreven om de joystick mee te kunnen testen. Ik heb deze maand ook meer opzoek werk vericht over de andere componenten.


Oktober

In oktober zijn we begonnen met onze gip. In deze maand heb ik vooral opzoek werk gedaan over de componenten die ik zal moeten gebruiken bij mijn gip. Maar ik heb ook een planning gemaakt aan de hand van het programma gantproject.
